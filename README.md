# Agent Fleet

A workforce of agents to handle many tasks, built using the [Mastra](https://mastra.ai/) framework.

## Agents

### TravelAgent

Agent that can find affordable vacations and builds itineraries. Sends the final itinerary via email.

### CEOAgent

Agent that can find microSaaS opportunities: performs market research, builds landing pages to validate interest, and drafts up technical planning documents. Has four custom evaluation scorers that sample outputs at runtime to measure delegation quality and phase awareness.
